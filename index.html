常用命令

用户名和邮箱

  • 查看用户名邮箱
    $ git config user.name
$ git config user.email
  • 设置用户名和邮箱
    $ git config --global user.name "用户名或昵称"
$ git config --global user.email "邮箱"

初始化

$ git init

远程仓库

  • 添加一个远程仓库
    $ git remote add origin 仓库地址
