Markdown中插入图片——使用七牛云作为图床

发表于 2016-08-29   |   更新于 2016-08-30   |     |   阅读次数

背景

刚刚接触Markdown，发现使用Markdown写作插入图片非常麻烦，各种搜索最后确定使用七牛云作为图床，对于我来说免费的10G空间足够使用了，接下来便是另一个麻烦的问题：怎么在Markdown快速的插入图片，又是各种搜索。。。无意间发现了一款非常好用的Atom插件，妈妈再也不用担心我如何Markdown中插入图片了，soeasy！

准备

  • Atom
  • 七牛云账号

七牛云账号

可通过我的邀请链接注册七牛账号，登录之后点击添加资源中对象存储中的立即添加，设置空间名称，存储区域，访问控制为公开空间，最后点击确定创建，这样就建立好了一个存储空间。

七牛免费账户有10GB的存储空间，每月10GB的HTTP国内下载流量、10GB的HTTP海外下载流量、100万次GET请求和10万次PUT请求，一般用户足够使用了。

安装

安装Atom

到Atom官网下载安装Atom，非常好用的一款编辑器。

安装markdown-assistant插件

打开Atom，按Ctrl+,打开设置界面，点击install，在输入框中输入markdown-assistant回车搜索，点击install安装knijhtli的这款插件。

安装qiniu-uploader插件

同样的，搜索qiniu-uploader，点击install进行安装。

配置

打开设置，在packages中找到qiniu-uploader,点击settings

按照提示填入相应的内容即可。

注意，空间名称即为上面设置的空间名称，域名我使用的是测试域名，因为是免费账户。

配置好之后就可以愉快的插入图片啦，

  1. 截图或复制文件
  2. Ctrl+V
  3. 填入title
  4. 回车

搞定！
